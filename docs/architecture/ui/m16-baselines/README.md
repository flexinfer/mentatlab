# M16 UI Baselines

This directory contains the visual baseline references for the M16 release of MentatLab Mission Control. These screenshots serve as the regression testing baseline for visual tokens, layout, and component chrome.

## Required Baselines

Per M16.8 requirements, please capture and place the following screenshots in this directory:

- `mission-control-empty.png`: Mission Control with an empty canvas.
- `active-runs-panel.png`: Active runs panel with mock data.
- `streaming-canvas-session.png`: Full streaming canvas session with active node states.

> **Note:** These baselines should be captured after all M16.1 through M16.7 UI standardization tasks have been merged to main.
